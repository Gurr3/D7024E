This is a project for the course D7024E at Luleå University of Technology (ltu.se). The purpose of this project is, to quote the lab description:

"
The objective for this proposal is to develop a minimalistic Platform-as-aService (PaaS) 
for deploying and executing Node.js based web applications.

Important: The PaaS should be based on OpenStack and Docker.
                                                                "
